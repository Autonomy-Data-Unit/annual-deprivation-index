ADI chart colour language

Token ramps applied to chart roles. Neutral slate carries the data surface; golden is reserved for the one highlighted/selected element. Furniture (axes, gridlines) stays faint — data-ink first.

Sequential — choropleth

Cool slate, pale→charcoal. The map data surface.

Diverging — change maps

Teal (improvement) ↔ rust (worsening). Not golden.

Line — COVID claimant spike

National UC claimant rate 2014–2024. Spike band 2019→2020; one series in ink, England mean as faint reference.

Bar — domains vs IMD correlation

r vs IMD per domain. Selected domain golden; others ink.
